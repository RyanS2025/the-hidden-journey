# The Hidden Journey

## Project Structure

```
the-hidden-journey/
├── docs/                    # All documentation
│   ├── architecture/        # Architecture and design documents
│   └── *.md                 # PRDs and other project docs
├── mockups/                 # UI mockups and design samples
├── tests/                   # All test files
└── CLAUDE.md
```

## Conventions

- **PRDs and project docs** go in `docs/`
- **Architecture documents** go in `docs/architecture/`
- **UI mockups and design samples** go in `mockups/`
- **Tests** go in `tests/`
